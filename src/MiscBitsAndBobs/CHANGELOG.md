# Changelog

## 2.3.7 | 19 May 2026

- Fixed a launch crash on the 32-bit GOG build with certain mod combinations
- Mod settings menu (F1) labels now follow your in-game language

## 2.3.6 | 3 May 2026

- Improved diagnostic logging for bug reports

## 2.3.5 | 27 April 2026

- Fixed Chinese translations not loading
- Added a main-menu notice when a newer version is on Nexus. Toggle off in settings if you don't want it
- Settings menu reorganised. Existing values preserved.

## 2.3.4 | 15 April 2026

- Fixed "Remove Cinematic Letterboxing" not working

## 2.3.3 | 12 April 2026

- Fixed non-English translations not loading
- Language changes in the game options are now picked up immediately without needing to restart
- Enabling Debug logging now shows a one-time in-game dialog warning you it's on

## 2.3.2 | 11 April 2026

- Translations are now loaded from editable JSON files in the lang folder
- Fixed several translation errors across multiple languages
- Main menu now shows "BepInEx Modded" in the version text
- Added a player judder fix that smooths out stuttering when moving
- Capped maximum footprints to prevent slowdown over long sessions
- Added configurable footprint fade speeds for outside, inside, and rain
- Fixed footprints popping out of existence instead of fading smoothly
- Added an option to force stuck church visitors to leave
- Improved compatibility across different BepInEx versions

## 2.3.1

- Fixed compatibility issue with Pray the Day Away
- GYK Helper is no longer required

## 2.2.8

- Fixed/added option for mute on alt tab, and allow the game to run in the background
