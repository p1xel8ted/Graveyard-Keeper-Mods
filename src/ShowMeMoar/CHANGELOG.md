# Changelog

## 0.1.13 | 19 May 2026

- Fixed a launch crash on the 32-bit GOG build with certain mod combinations
- Mod settings menu (F1) labels now follow your in-game language

## 0.1.12 | 3 May 2026

- Improved diagnostic logging for bug reports

## 0.1.11 | 27 April 2026

- On first launch with Windows display scaling above 100%, the mod offers to apply the high-DPI fix automatically. Restart after accepting. Apply, undo, or silence the prompt under the new High-DPI Fix settings section.
- Corrected the manual high-DPI workaround on the mod page: set "High DPI scaling override" to "Application" instead of "System". If the menu still looks blurry after restarting, open in-game Settings → Display and change the resolution once to snap it to the correct DPI. The "try the Zoom slider" tip has been removed.
- Added a main-menu notice when a newer version is on Nexus. Toggle off in settings if you don't want it

## 0.1.10 | 11 April 2026

- Fixed hotkeys and input features not working for some users
- Improved compatibility across different BepInEx versions

## 0.1.9

- Added borderless windowed mode option
- Added ultrawide fog fix for displays wider than 2880px
- GYK Helper is no longer required

## 0.1.6

- Option to toggle color correction
- Main menu artwork fixed. Looks fine on 21:9, not so sure about 32:9+. No change at 16:9.
