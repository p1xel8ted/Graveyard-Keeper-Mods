# Changelog

## 2.1.15 | 19 May 2026

- Fixed a launch crash on the 32-bit GOG build with certain mod combinations
- Mod settings menu (F1) labels now follow your in-game language

## 2.1.14 | 3 May 2026

- Fixed zombies on human workbenches dropping crafted items on the ground instead of into a nearby trunk
- Added +1 and -1 buttons to the expanded multi-quality craft view
- Improved diagnostic logging for bug reports

## 2.1.13 | 27 April 2026

- Fixed Chinese translations not loading
- Fixed graves, garden beds, and other build-menu items becoming unplaceable when Auto Max Normal Crafts or Auto Max Multi-Quality Crafts was enabled
- Fixed "Auto Select Craft Button With Controller" not focusing Craft Now on multi-quality crafts
- New installs now have all workbench categories enabled by default. Existing installs keep your current picks.
- Added a main-menu notice when a newer version is on Nexus. Toggle off in settings if you don't want it
- Rest In Patches is now a required dependency. Install it first or this mod will refuse to load

## 2.1.12 | 15 April 2026

- Setting changes now apply immediately without a game restart.
- Auto-crafting is now opt-in per workbench under the new Auto-Craft section. Everything starts off.
- Sermons at the church pulpit no longer run on their own unless you enable Auto-Craft → Sermons.
- Renamed "Make Everything Auto" to "Enable Auto-Craft" as the top-level switch.
- Retired the old "Make Hand Tasks Auto" option. The per-workbench toggles replace it.
- Added a Balance section with a "Half Research Point Outputs" toggle, on by default. Turn it off for full yields from auto-crafted recipes.
- Settings menu reorganised. Existing values preserved.
- Cleaned up the auto-craft target list - many decor, repair, quarry, and soul benches no longer auto-craft.

## 2.1.11 | 12 April 2026

- Enabling Debug logging now shows a one-time in-game dialog warning you it's on
- Fixed non-English translations not loading
- Language changes in the game options are now picked up immediately without needing to restart

## 2.1.10 | 11 April 2026

- Fixed hotkeys and input features not working for some users
- Improved compatibility across different BepInEx versions

## 2.1.9

- Fixed auto-craft interfering with zombie workers and dropping items on the ground
- Fixed wine barrels and similar recipes breaking when Auto Max Multi-Quality Crafts was enabled
- GYK Helper is no longer required
